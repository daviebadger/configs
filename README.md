# Davie's dotfiles

My personal configuration files for [Ubuntu Budgie 18.10][Ubuntu Budgie] desktop.

[Ubuntu Budgie]: https://ubuntubudgie.org/

## Installation

```
$ wget -0 - https://raw.githubusercontent.com/daviebadger/dotfiles/master/install.sh | bash
```
